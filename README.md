# IIT Short Course

This repository contains practice materials for a beginner Python course. It includes basic Python scripts and exercises for `Class-5`.

## Contents

- `Class-5/`
  - `1_basic.py` - Basic Python concepts and syntax
  - `2_simpleCountdown.py` - A simple countdown loop
  - `3_sum_calculator.py` - A program to add numbers
  - `4_input_validator.py` - Input validation logic
  - `5_break.py` - Using `break` to exit a loop
  - `6_continue.py` - Using `continue` inside a loop
  - `7_pssword.py` - A simple password checker
  - `Exercise/` - Practice exercises
    - `1_countdown.py` - Countdown practice
    - `2_even.py` - Find even numbers
    - `3_positive.py` - Identify positive numbers
    - `4_guess_number.py` - Guess the secret number
    - `5_multiplication.py` - Generate multiplication tables
    - `6_power.py` - Calculate powers

## How to run

1. Make sure Python is installed on your system.
2. Open a terminal or command prompt.
3. Go to the repository root:

```bash
cd d:\ShortCourse\IIT-short-course
```

4. Run any script like this:

```bash
python Class-5\Exercise\4_guess_number.py
```

## Tips

- For files that use `input()`, run them from the terminal and enter values when prompted.
- This project is educational, so open each file and read the comments to understand the code.

## Project goals

- Learn basic Python concepts
- Practice loops, conditions, and input handling
- Build confidence with small Python scripts
